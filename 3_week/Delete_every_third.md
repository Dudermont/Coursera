# Удалить каждый третий символ

>Дана строка. Удалите из нее все символы, чьи индексы делятся на 3.Символы строки нумеруются, начиная с нуля.
>   
## Формат ввода

 Вводится строка.

## Формат вывода

 Выведите ответ на задачу.

## Формат вывода

Ввод и вывод осуществлять с помощью файлов

 ## Примеры
>
>### **Тест 1**
>
>>*Входные данные:*

>>
>> Python

>>*Вывод программы:*

>>`yton`
